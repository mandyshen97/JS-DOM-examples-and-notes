### 用 JavaScript  DOM 实现一个图片库的功能

- 如何利用 DOM 所提供的方法去编写图片库脚本。

- 如何利用事件处理函数把 JavaScript 代码与网页集成在一起。

---
用到的：
`getElementById()`
`getElementsByTagName()`
`getAttribute()`
`setAttribute()`
`childNodes`
`firstChild`
`nodeValue`
`nodeType`
`appendChild()`
`insertAfter()`

#### DOM 脚本编程有关的问题：
- **平稳退化**   
- **分离 JavaScript**
- **向后兼容**   
    + 对象检测
- **性能考虑**
    + 压缩脚本
    + 尽量减少 DOM 和尽量减少标记
    + 合并和放置脚本
